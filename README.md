# Email Cron Job
A small cron job that sends scheduled emails automatically.

## Requirements

- Java 17 / Spring Boot 3 (if using Spring)
- SMTP server credentials
- Maven or Gradle

- ## How to Run
1. Clone the repository:
2. Configure SMTP credentials in `application.properties`:
3. Build and run:

## Cron Schedule
This job is scheduled to run **every 30 minutes**.  

## Notes
- Make sure your SMTP server allows less secure apps if needed.
- Logs are printed to the console for simplicity.
